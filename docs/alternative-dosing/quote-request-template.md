# Quote-request email template

Reusable template for the five products tracked in
[`README.md`](README.md). Replace the bracketed placeholders before
sending. Keep the body identical across vendors so the returned quotes
are directly comparable.

---

**Subject:** Request for quotation — [PRODUCT NAME] (academic R&D
deployment)

Hello [VENDOR CONTACT],

I'm writing on behalf of the Vertical Cloud Lab to request a formal
quotation for **[PRODUCT NAME / SKU]** (product page:
[URL]). We are evaluating commercial powder-dosing instruments as part
of an academic R&D project on autonomous powder handling
(`vertical-cloud-lab/powder-doser`).

### Application

- **Lab type:** academic / self-driving-lab research group.
- **Powders:** ceramic and metal-oxide precursors plus a few cohesive
  organics (e.g. xanthan gum, alumina, lithium salts). Particle sizes
  span ~5–150 µm; some materials are static-prone.
- **Dose range:** ~1 mg – 50 g per vial.
- **Throughput target:** [N] samples per run, [M] runs per day.
- **Sample format:** [VIAL TYPE / SBS RACK].
- **Environment:** [ambient bench / fume hood / inert glovebox —
  specify].
- **Software:** we need a scriptable interface (Python / REST / CSV
  import-export) so the instrument can be driven from our host
  workflow.

### Please itemise the quote so we can compare vendors

1. Base instrument and all standard accessories.
2. Dosing heads / feeders covering the powders above (please advise).
3. Required PC, balance, and any consumables for the first 3 months.
4. Glovebox / enclosure / anti-static add-ons.
5. Installation, on-site commissioning, and operator training.
6. 1st-year warranty plus options for an extended service plan.
7. Freight, insurance, and applicable taxes/duties to:
   [SHIP-TO ADDRESS].

### Other questions

- **Lead time** — please confirm both stocking lead time and any
  build-to-order lead time for the configuration above.
- **Compliance** — CE certification is sufficient for our site;
  please flag if NRTL/CSA is required and at what additional cost.
- **Quote validity** — please make the quote valid for at least 60
  days.
- **References** — if available, we'd appreciate the names of one or
  two academic groups using the same configuration.

Thank you — happy to set up a 30-minute call if it helps you scope the
configuration.
