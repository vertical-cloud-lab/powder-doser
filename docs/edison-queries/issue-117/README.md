# Edison literature queries — Issue #117 (powder dosing for synthetic chemistry)

Low-effort Edison Scientific (PaperQA) literature queries on the powder-handling
properties of three reagents proposed by Phillip Lampkin (University of Utah) as
test cases for milligram-scale (0.1–10 mg) auger-based dosing. See
[Issue #117](https://github.com/vertical-cloud-lab/powder-doser/issues/117) for context.

| Reagent | CAS | Report |
|---|---|---|
| AIBN (azobisisobutyronitrile), frontal polymerization initiator | 78-67-1 | [aibn-78-67-1.md](aibn-78-67-1.md) |
| Grubbs G2 Ru metathesis catalyst (glovebox dosing candidate) | 246047-72-3 | [grubbs-g2-246047-72-3.md](grubbs-g2-246047-72-3.md) |
| Sodium cyanide (hands-off dosing of toxic solids) | 143-33-9 | [sodium-cyanide-143-33-9.md](sodium-cyanide-143-33-9.md) |

Each report includes the exact question asked, the full cited answer, and the
reference list. A recurring caveat across all three: direct powder-characterization
data (bulk/tapped density, PSD, shear-cell flowability, tribocharging) is largely
unpublished for these specific compounds, so lot-specific measurement is
recommended before qualifying a dosing process.
