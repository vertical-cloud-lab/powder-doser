# Auger slices (v4 — integrated single-piece rotor, H2D)

The auger is **one printable part** (per PR review reversion of the v3
two-part experiment back to the v1/v2.1 monolithic architecture). It is
sliced for the **Bambu Lab H2D** (build volume 350 × 320 × 325 mm —
the 250 × 25 mm pillar has ~75 mm of Z headroom and fits the bed with
plenty of room for brim and skirt) on 1.75 mm PLA at 0.4 mm nozzle /
0.2 mm layers / 3 perimeters / 40 % gyroid infill / 4 mm brim /
supports at 50°.

## Files

### PrusaSlicer CLI (canonical)

| Printer | G-code | Time | Material |
|---|---|---|---|
| Bambu Lab H2D | [`archimedes-auger.H2D.gcode`](archimedes-auger.H2D.gcode) | ≈ 7 h 31 m | 73.5 cm³ (~92 g) |
| Bambu Lab H2D (USB short name) | [`AUGER.gcode`](AUGER.gcode) | same as above | same as above |

PrusaSlicer's Marlin (legacy) g-code flavour is what the H2D's
RepRap-derived firmware expects once start/end blocks are spelled out
explicitly (the script writes the resolved blocks — no `{placeholder}`
substitution required).

### CuraEngine (Ultimaker Cura's open-source slice engine)

Generated by `slice_cura.sh` when `cura-slicer` is available on the
runner. Same naming pattern with `.cura.gcode` suffix:

```
archimedes-auger.H2D.cura.gcode
```

## Print orientation & supports

Print **exit-hole DOWN on the bed** (Z+ = M3 boss). The 1.5 mm exit hole
sits in the centre of a small annular bed contact — 4 mm brim is enough
adhesion. Supports at 50° handle:
  * the helix's inner-edge bridge above the funnel cavity, and
  * the top cap's boss-to-wall annulus bridge at the very top.

After printing:
  * Tap the M3 spindle hole with an M3 hand tap.
  * If the exit hole comes out tight (< 2.5 mm functional ID), open it
    with a 2.5 mm drill bit.

## Step-by-step on the printer

1. Copy `AUGER.gcode` to the H2D via USB / SD card / cloud upload.
2. Pre-flight: bed levelled, nozzle clean, PLA loaded. The file sets
   PLA 215/215 °C and bed 60 °C in its start block, so don't override
   on the LCD before pressing start.
3. The start block is `G28` → `G1 Z5 F5000`: home, then lift 5 mm, then
   begin layer 1.
4. ≈ 7.5 h print time — start before a workday, finish during.
